AUTOMATIC TUTORIAL MAKER

Quick start guide for Unity developers by Octanta Studio

Automatic Tutorial Maker creates game tutorials through automatic recording of developer actions or manual step creation. Record once, customize later.

Multiple input types (click, swipe, hold, etc)
PC, mobile, gamepad support
2D, 3D, UI objects recognition
Automatically or manually called steps

LEARNING SEQUENCE EXAMPLE

# Start Step → Auto After Previous Step
M Start Step → Manual Call

Edit Step Description

SETUP

  • Window → Package Manager → Packages: My Assets → Import AutomaticTutorialMaker
  • Add the TutorialSystem prefab to your scene
  • Right click → Prefab → Unpack
  • Select TutorialSystem → TutorialSceneReferences
  • Assign Main Camera and Target Canvas

CREATING TUTORIAL FOR SCENE

RECORDING
  • Start Play Mode
  • Select AutomaticTutorialMaker
  • Click Start Recording
  • Perform the necessary actions (clicks, swipes, typing)
  • Track tutorial steps being created in real time
  • Click Stop Recording
CUSTOMIZATION
  • Edit in ATM (remove unnecessary steps, search for objects by tag)
  • Start Play mode again
  • Check how the hints work
  • Reset tutorial progress if needed (StepSequencePlayer → Reset)
