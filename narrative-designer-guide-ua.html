SETUP

  • Додайте NarrativeDesigner.prefab до вашої сцени
  • Створіть необхідні компоненти, як зазначено в меню
CHARACTER BRAIN
Характер персонажа, навички, спогади
CHARACTER DATABASE
Часова шкала і стосунки персонажів
GLOBAL CONTEXT
Лор, механіки, медіа
ЛОКАЛЬНІ КОНТЕКСТИ (ПОДІЇ НА ШКАЛІ ЧАСІ)
Натисніть на подію, щоб переглянути деталі. Перетягуйте події, щоб міняти їх порядок на часовій шкалі.

МАТРИЦЯ ПЕРСОНАЖІВ

  • Натисніть 'Open Full Matrix View', щоб побачити глобальну часову шкалу
  • Увімкніть 'Descriptive View' для текстових описів або 'Numeric View' для значень впливу
  • Чорно-біла шкала показує конструктивні та деструктивні стани персонажів під час цих подій
Деструктив (-100) Конструктив (100) 0 = Відсутність знань про подію
СТВОРЕННЯ ПОДІЙ НА ЧАСОВІЙ ШКАЛІ
  • Відкрийте об'єкт Character Database
  • У розділі Selected Event введіть опис вашої сюжетної події
  • Перетягніть спрайт у поле ілюстрації
  • У розділі Character Effects встановіть, як ця подія впливає на кожного персонажа (від -100 до +100)
  • Розгорніть знак '+' поруч із кожним іменем персонажа, щоб додати деталі конкретної поведінки
  • Натисніть 'Save Event As New', щоб додати її до загальної часової шкали
КЕРУВАННЯ ПОДІЯМИ
  • Усі створені події з'являються у списку та матриці подій часової шкали під областю редагування
  • Натисніть на будь-яку подію, щоб завантажити та редагувати її деталі
  • Перетягуйте події, щоб змінювати їх порядок на часовій шкалі

СТРАТЕГІЧНА ДОШКА

  • Перетягуйте персонажів, щоб змінювати їхні позиції
  • Натискайте на лінії між персонажами, щоб редагувати взаємовідносини
  • Використовуйте різні кольори та розміри крапок, щоб позначати статус персонажів
  • Встановіть власну фонову мапу ігрового світу
