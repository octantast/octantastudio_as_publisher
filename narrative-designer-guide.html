Narrative Designer is designed to visualize and systematize narrative elements and get AI help with story development.

SETUP

  • Import the asset package into your Unity project
  • Find Assets > OctantaStudio > NarrativeDesigner > NarrativeDesigner.prefab and add it to your scene
  • Create required components as indicated in the menu

CREATE TIMELINE EVENTS
  • Open the Character Database object
  • In the Selected Event section, enter a description of your story event
  • Drag a sprite into the illustration field if available
  • Under Character Effects, set how this event impacts each character (-100 to +100)
  • Expand the '+' next to each character name to add specific behavior details
  • Click 'Save Event As New' to add it to the global timeline
MANAGE TIMELINE EVENTS
  • All created events appear in the Timeline Events list & matrix below the edit area
  • Click on any event to load and edit its details
  • Drag events to reorder them in the timeline

CHARACTER MATRIX

  • Click 'Open Full Matrix View' to see a global timeline
  • Use 'Descriptive View' for text descriptions or 'Numeric View' for impact values
  • Black-and-white scale shows constructive vs. destructive character states during these events
Destructive (-100) Constructive (100) 0 = No knowledge of event

STRATEGIC BOARD

  • Drag characters to rearrange their positions on the board
  • Click on lines between characters to edit relationships
  • Use different colors and dot sizes to indicate character status
  • Set custom background map of your game world
